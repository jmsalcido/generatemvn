generatemvn

If you want to create a fast mvn project you only need to lookup the archetype at google: http://docs.codehaus.org/display/MAVENUSER/Archetypes+List

It is easy to install (_like... really easy_) just:

1.- Download the script:

curl https://raw.github.com/jmsalcido/generatemvn/master/generatemvn > generatemvn

2.- Just need to chmod the file:

chmod +x generatemvn

3.- And put it anywhere in $PATH:

mv generatemvn ~/bin # I use the bin folder in my user folder for user scripts / programs.

Why does this exist?

Well I dont remember the mvn command to generate a project always (_and those arguments..._), so I built this small and ugly script for some help.
